# Fizz Buzz Tree

## Summary
- Create a method that takes a binary tree as an argument, without utilizing any of the built-in methods, 
determine weather or not the value of each node is divisible by 3, 5 or both, and change the value of each of the nodes respectively. 
Return the tree with it’s news values.
