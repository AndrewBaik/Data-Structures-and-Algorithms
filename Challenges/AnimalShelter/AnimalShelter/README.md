# FIFO Animal Shelter

## Challenge Summary
- AnimalShelter which holds only dogs and cats. The shelter operates using a first-in, first-out approach.
- Implement Enqueue(animal), adds animals(dogs or cats) animals to shelter.
- Implement Dequeue(pref), returns either a dog or a cat. If pref, a string, is ‘cat’ return the longest-waiting cat. If pref is ‘dog’, return the longest-waiting dog. For anything else, return either a cat or a dog.
